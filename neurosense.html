NeuroSense

AI-powered real-time stress detection & productivity Management system

AI SYSTEM ACTIVE

Overview

NeuroSense is an AI-powered multimodal stress detection system that analyzes facial expressions, voice recordings, and text input to detect stress levels in real time and the system provides personalized stress management recommendations while offering a secure, user-friendly platform for continuous mental well-being monitoring

What it does

  • Engineered a real-time multimodal stress detection system analyzing video, audio, and text using FastAPI, React, DeepFace, MediaPipe, and Google Gemini.
  • Built multimodal pipelines for facial emotion recognition, speech-based emotion analysis, and contextual sentiment detection.
  • Shipped an interactive dashboard with real-time analytics and personalized wellness recommendations.

My role

I was primarily responsible for the Text Analysis Module and the integration of the frontend and backend in the NeuroSense project. I developed the text analysis workflow by integrating the Gemini API to analyze user text and identify emotions, sentiment, and stress-related patterns. I also connected the React (Vite) frontend with the FastAPI backend using REST APIs, ensuring smooth communication between the user interface and the server. Additionally, I participated in API testing, debugging, system integration, and validating the application's functionality to ensure accurate and reliable performance.
